How this works ?

Its a dance of Golang, AWS, Let's Encrypt and S3

19 Oct, 2020

When I decided to start sharing something publicly I took months to choose on which would be the approach that would fit my goal... should I do GitHub Pages, Medium bring up a Wordpress or some staticly generation tool like jenkyll or hugo. As a core software engineer and a passion to craft I chose the not so straight forward path and a more laborious one.

This site is a tiny golang application whose you can find here, the content is served from an aws ec2 micro instance. as well know it has only 1GB of RAM and something more than this of storage. which I do not use. The posts are fetched from S3, I also avoid requesting s3 for every made request... It uses PureCSS and Microdata Schema
